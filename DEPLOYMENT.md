## Update Flow

1. Make edits here.
2. Open GitHub Desktop.
3. Review the changed files.
4. Commit to `main`.
5. Push origin.
6. Wait for GitHub Pages to redeploy.

GitHub Pages does not update from local edits alone. It updates after a commit is pushed to GitHub.
